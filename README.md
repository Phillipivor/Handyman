# Handyman Service App

A React Native/Expo application for connecting users with handymen and service providers.

## Features

- Multi-user roles (User, Handyman, Service Provider, Admin)
- Real-time chat
- Service booking system
- Job management
- Payment processing
- Photo uploads
- Location services

## Setup

```bash
npm install
npm start
```

## Tech Stack

- React Native / Expo
- React Navigation
- Context API
- Axios
